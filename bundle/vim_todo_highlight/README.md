vim-highlight
=============

Plugin implements a number of simple regex matches to create highlights.

Will highlight:
- The word 'Note', followed by either a colon (:) or whitespace yellow
- The word 'Todo', followed by either a colon (:) or whitespace yellow
- The extra characters in any line over 79 characters with a grey background
- Any whitespace at the end of the line as red in the following files:
  - c
  - cpp
  - html
  - ruby
  - perl
  - php
  - python
  - python3
